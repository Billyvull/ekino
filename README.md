# 🎬 Ekino — Movie Website

Веб-сайт для просмотра и управления каталогом фильмов, разработанное на PHP с использованием базы данных MySQL.

Проект включает пользовательскую часть сайта и административную панель для управления контентом.

## 🚀 Возможности проекта

### Пользовательская часть:

* просмотр каталога фильмов
* страницы с информацией о фильмах
* разделение фильмов по жанрам
* поиск фильмов
* просмотр описания и информации о фильме
* адаптивный интерфейс

### Административная панель:

* добавление новых фильмов
* редактирование информации о фильмах
* удаление фильмов
* управление контентом сайта

## 🛠 Используемые технологии

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## 🗄 База данных

Для хранения информации используется MySQL.

В базе данных хранятся:

* фильмы
* жанры
* описание фильмов
* изображения
* данные для управления сайтом

## ⚙️ Функциональность администратора

Администратор может управлять каталогом фильмов через отдельную панель управления:

* создавать новые записи
* изменять существующие данные
* обновлять информацию на сайте

## 📌 О проекте

Проект создан для практики backend-разработки на PHP и работы с реляционными базами данных.

Основной акцент сделан на:

* взаимодействие frontend и backend
